# Ore Sheep [![Curseforge](https://cf.way2muchnoise.eu/full_226876_downloads.svg)](https://www.curseforge.com/minecraft/mc-mods/ore-sheep-mod)

This mod adds ore sheep. Convert regular sheep to grow ore!
